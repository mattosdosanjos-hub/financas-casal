Como conectar ao Google Sheets

Este app precisa de uma URL do seu Apps Script para enviar os lançamentos. Siga os passos:

1 Abra a planilha → Extensões → Apps Script
2 No editor, clique em Implantar → Nova implantação
3 Tipo: App da Web → Executar como: Eu → Quem tem acesso: Qualquer pessoa
4 Clique em Implantar e copie a URL gerada
5 Cole a URL abaixo e salve

Instalar na tela inicial (iOS)

Safari → botão compartilhar ⬆ → Adicionar à Tela de Início

Instalar na tela inicial (Android)

Chrome → menu ⋮ → Adicionar à tela inicial
